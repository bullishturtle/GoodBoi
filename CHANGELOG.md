# Changelog

## [1.0.0] - 2025-11-26

### Added
- Complete council architecture with 6 specialized agents
- Self-learning and evolution system
- Memory management with conversation history
- Local GGUF model support via GPT4All
- Model Manager for downloading models
- Desktop UI with Tkinter
- Web dashboard with Gradio
- FastAPI backend with full REST API
- Teachings store for continuous learning
- Mini-bot nursery for spawning specialized sub-agents
- Automatic routing optimization
- Windows installer via Inno Setup
- Comprehensive documentation

### Features
- **4 Routing Modes**: Auto, Reflex, Council, Strategic
- **Memory System**: Persistent conversation history
- **Evolution Tracking**: Generations and proficiency metrics
- **Self-Reflection**: Performance analysis and optimization
- **Tool Integration**: Extensible tool system for agents
- **Security Layer**: Audit logging and safe operation
- **Cloud/Local**: Switch between cloud API and local models

### Agents
- **Batman**: Strategy & Security
- **Alfred**: Scheduling & Communication
- **Jarvis**: System Control (core)
- **DaVinci**: Creativity & Design
- **Architect**: Builder & Code
- **Analyst**: Data & Insights
