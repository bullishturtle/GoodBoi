# GoodBoy.AI Launch Guide

## Quick Start (Windows)

### Option 1: Automatic (Recommended)

1. **Install**:
   \`\`\`
   INSTALL.bat
   \`\`\`

2. **Launch**:
   \`\`\`
   GoodBoy_launcher.bat
   \`\`\`

### Option 2: Quick Diagnostic

\`\`\`
QUICKSTART.bat
\`\`\`

This will check your installation and offer to launch automatically.

### Option 3: Manual Setup

```batch
# Install dependencies
python install.py

# Start server
run_server.bat

# Start UI (in another window)
run_desktop_ui.bat
