GoodBoy.AI Installation Complete!

GETTING STARTED:
1. Launch GoodBoy.AI from your desktop or Start menu
2. The backend server will start automatically
3. Download an AI model in Settings > Model Manager
4. Start chatting!

RECOMMENDED MODELS:
- GPT4All Falcon (Small, 4GB RAM)
- Mistral 7B Instruct (Medium, 8GB RAM)
- WizardCoder 13B (Large, 16GB RAM)
