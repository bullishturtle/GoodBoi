Welcome to GoodBoy.AI Installation

GoodBoy.AI is a self-learning, autonomous AI assistant that operates locally on your computer for maximum privacy and control.

SYSTEM REQUIREMENTS:
- Windows 10/11 (64-bit)
- Minimum 8GB RAM (16GB recommended)
- 5GB free disk space (more for AI models)
- Python 3.10+ (optional, for advanced features)

WHAT YOU'LL GET:
- GoodBoy Desktop UI (main application)
- GoodBoy Backend Server (council architecture)
- Model Manager (download and manage AI models)
- Memory System (learns from your interactions)
- Dashboard (monitor AI performance)

FIRST RUN:
1. Launch GoodBoy.AI
2. Download an AI model using the Model Manager
3. Start chatting - GoodBoy will learn and improve over time

This installer will set up everything you need to run GoodBoy.AI.
